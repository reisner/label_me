# Label-Me

Small Rails app for labelling images. 

## Setup

* `bundle install`
* `rails db:create`
* `rails db:migrate`
* Verify tests pass: `rspec`
* Run the dev server: `bin/dev`
* To use your own labels, add them (one per line) to `config/labels.txt`
